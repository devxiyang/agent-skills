# Common Mistakes

## 1. Wall of text

**Problem:** Entire paragraphs copied onto a slide. Audience reads ahead, stops listening.

**Fix:** Extract the one sentence that matters. Everything else belongs in speaker notes or a document.

Before:
> Our platform leverages cutting-edge machine learning algorithms to process real-time data streams, enabling businesses to make informed decisions faster than ever before while reducing operational costs significantly.

After:
> **3× faster decisions.** 40% lower cost.

---

## 2. Title that names the topic, not the point

**Problem:** "Q3 Results" — tells the audience nothing.

**Fix:** Write the conclusion. Make the title the thing you'd say if you only had 5 seconds.

| Weak | Strong |
|---|---|
| Q3 Results | Revenue up 28% — best quarter ever |
| Competitive Analysis | We're cheaper and faster than all three competitors |
| Next Steps | Ship by Friday or lose the contract |

---

## 3. Too many bullets

**Problem:** 8 bullets per slide. Eyes glaze over, nothing is memorable.

**Fix:** Maximum 5 bullets. If you have 8 points, make 2 slides. One idea per bullet, one takeaway per slide.

---

## 4. Inconsistent visual style

**Problem:** Each slide uses different font sizes, alignments, or colors. Looks like a draft.

**Fix:** Never override the layout classes with inline styles unless you have a specific reason. The CSS system handles consistency automatically.

---

## 5. Using color for decoration

**Problem:** Random colored headings, gradient backgrounds per slide, rainbow bullet points.

**Fix:** Color means something. Accent color = the most important thing on this slide. Use it once per slide, maximum twice.

---

## 6. Tiny font to fit more content

**Problem:** Font shrunk to 14px to squeeze in 200 words.

**Fix:** If text needs to be small to fit, remove it. No one reads 14px text on a projected screen.

---

## 7. Missing a clear ending

**Problem:** Last slide says "Thank You" or "Q&A." Audience doesn't know what to do next.

**Fix:** End with a specific ask: approve the budget, sign up for the beta, schedule a follow-up. Make it one sentence.

---

## 8. Animations on every element

**Problem:** Every bullet flies in from a different direction. Exhausting to watch.

**Fix:** In this template, the only animation is a 0.3s fade between slides. That's enough. If you want to reveal bullets one by one, use separate slides.

---

## 9. No contrast between slides

**Problem:** All slides look identical — same layout, same density, same color weight.

**Fix:** Vary the layouts intentionally. Put a `layout-quote` or `layout-title` between heavy content slides. Contrast in rhythm keeps attention.

---

## 10. Presenting the deck instead of the story

**Problem:** Reading bullet points aloud. The deck is the presentation.

**Fix:** The deck supports the story; it doesn't replace it. Each slide should prompt what you say, not replace it.
