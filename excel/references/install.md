# Installing Excel dependencies

Requires Python 3 and two packages: `openpyxl` (read/write xlsx) and `pandas` (data transformation).

## Install Python packages

```bash
pip install openpyxl pandas
```

Or with pip3:

```bash
pip3 install openpyxl pandas
```

## Install Python 3

### macOS

```bash
brew install python
```

### Windows

Download from https://www.python.org/downloads/ and run the installer. Make sure to check "Add Python to PATH".

### Linux

```bash
# Debian/Ubuntu
sudo apt install python3 python3-pip

# Fedora
sudo dnf install python3 python3-pip

# Arch
sudo pacman -S python python-pip
```
